¿Quién inventó el sistema de control de versiónes y por qué? 
GGit (pronunciado «guit»/gɪt​) es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.
