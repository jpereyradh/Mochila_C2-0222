a. ¿Quién inventó el sistema de control de versión Git y por qué?
    Git fue creado por  Linus Torvalds para llevar un control de versiones de aplicaciones cuando tienen muchos archivos de código fuente.

quien lo compro por la suma de 7500 millones de dólares.
