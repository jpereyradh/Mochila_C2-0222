a. ¿ Quien invento el sistema de control de verison Git y por que?
Le genial plataforma GitHub fue creada por Chris Wanstrath , PJ Hyett , Tom Preston-Werner y Scott Chacon ayá por Febrero de 2008 en San Francisco – EEUU y hasta la fecha sigue estando ubicada en el mismo lugar. La plataforma te permite trabajar llevando un control de las versiones de tu proyecto.
b ¿ A quien pertenece actualmente github y por que?
El 4 de Junio de 2018 Microsoft anuncia que hizo un acuerdo con Github para comprarlo por 7.5 mil millones de dolares, esta compra se cierra el 26 de Octubre de 2018. El 28 de Febrero de 2018 Github sufre un ataque DDoS, el segundo mayor ataque en su historia, sufrió un pico de trafico de 1.35 terabits por segundo.
