a. ¿Quién inventó el sistema de control de versión Git y por qué?  
El sistema fue desarrollado por Linus Torvalds. Porque pensaban en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando tienen un gran numero de archivos de código fuente.
