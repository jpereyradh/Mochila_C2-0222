
Hardware: Es el conjunto de elementos físicos y tangibles de un sistema informaticos. 
Existen 3 tipos de hardware: 
1. Dispositivos de entrada.
2. Dispositivos de salida.
3. Internos.

Software: Es el conjunto de elementos lógicos e intangibles que hacen parte del sistema
 informático que periten la ralización de tareas específicas. Existen tres tipos:
1. Software de sistema.
2. Software de aplicación.
3. Software de programación.
